# REBND
Clothing Store eCommerce/IMS Website Project

Requirements:

- Visual Studio Code (or any other source-code editor/IDE that supports PHP, HTML, SQL, CSS)
- XAMPP (Open Source Web Server)
- MySQL (Open Source Relational Database Management System)

How-To:

- Install XAMPP on your computer.
- Open XAMPP and start Apache and MySQL.
- Find location where xampp was installed in your computer.
- In xampp folder there should be a folder called "htdocs", open that folder.
- Create a new folder and name it "REBNDshop".
- Move all the files you downloaded from git repository into that REBNDshop folder.
- You should now be able to view each of the php files from the git repository as a website using localhost.
- In google chrome (or whatever is your preferred web browser), type: "localhost/REBNDshop/" in the address bar.
- This should bring you to a directory that has all of the files from the REBNDshop folder that you created in the htdocs folder in your hard drive.
- You should be able to open any of the php files and view them as a website in your browser by clicking any of the php files in the directory.
- When making a change to any of the files you can view how the changes you made looks in real time by saving the code and refreshing the file in your web browser.
